# Real Estate Analytics Framework - Quick Start Guide

## 🚀 Get Started in 3 Minutes

### 1. Install Dependencies
```bash
pip install pandas numpy matplotlib seaborn plotly dash scipy scikit-learn
```

### 2. Run the Demo
```bash
python demo.py
```

```bash
python test_framework.py
```
